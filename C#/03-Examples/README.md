# Koodiesimerkit ja Harjoitukset

Tässä kansiossa on lisäesimerkkejä ja harjoituksia olio-ohjelmoinnin konsepteihin.

## Harjoituksia

### Harjoitus 1: Luokka ja Objekti
Luo `Person`-luokka, jolla on:
- Ominaisuudet: `Name`, `Age`, `Email`
- Metodit: `Introduce()`, `IsAdult()`

### Harjoitus 2: Kapselointi
Luo `BankAccount`-luokka, jossa:
- Saldo on yksityinen
- Voidaan tallettaa ja nostaa rahaa
- Validoidaan, ettei saldo mene negatiiviseksi

### Harjoitus 3: Perintä
Luo `Animal`-luokka ja sen aliluokat:
- `Dog`, `Cat`, `Bird`
- Jokaisella oma `MakeSound()`-metodi

### Harjoitus 4: Polymorfismi
Käytä edellisen harjoituksen luokkia ja luo metodi, joka käsittelee kaikkia eläimiä yhtenäisesti.

### Harjoitus 5: Rajapinnat
Luo `IDrawable`-rajapinta ja toteuta se useassa luokassa (`Circle`, `Rectangle`, `Triangle`).

## Vinkkejä

- Aloita yksinkertaisista esimerkeistä
- Kokeile koodia itse Visual Studio Code:ssa tai Visual Studiossa
- Muuta esimerkkejä ja katso mitä tapahtuu
- Yritä luoda omia esimerkkejä
