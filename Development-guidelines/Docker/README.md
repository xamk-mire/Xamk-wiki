# Docker

Tervetuloa Docker-oppimateriaaliin! Tämä materiaali käsittelee Dockerin perusteet, Dockerfile-tiedostojen luomisen, Docker Composen käytön sekä volumet ja verkot.

## Sisältö

### Perusteet
- [Docker-perusteet](Docker-Perusteet.md) - Mikä on Docker, asennus, käsitteet ja peruskomennot

### Dockerfile
- [Dockerfile](Dockerfile.md) - Dockerfile-syntaksi, instruktiot, multi-stage build ja parhaat käytännöt

### Docker Compose
- [Docker Compose](Docker-Compose.md) - Compose-syntaksi, palvelut, ympäristömuuttujat ja komennot

### Volumet ja Verkot
- [Volumet ja Verkot](Volumet-ja-Verkot.md) - Datan pysyvyys, verkot ja konttien välinen kommunikaatio

## Oppimisjärjestys

Suosittelemme opiskelua seuraavassa järjestyksessä:
1. **Docker-perusteet** - Ymmärrä käsitteet ja opi peruskomennot
2. **Dockerfile** - Opettele rakentamaan omia Docker-imageja
3. **Volumet ja Verkot** - Ymmärrä datan pysyvyys ja konttien välinen kommunikaatio
4. **Docker Compose** - Opettele hallitsemaan usean kontin sovelluksia
